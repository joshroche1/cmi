Cloud Management Interface

Description

This system is designed to perform simple administrative tasks on servers deployed to cloud providers such as Amazon Web Services EC2 and Google Cloud Platform. It can also work with server instances run on several hypervisor systems such as Citrix XenServer and VMWare ESXi.

Features:

  • Web Console for Administration
  • Monitor server instances and restart/shutdown systems
  • Update server systems through native package management
